  • avoid needless media queries

  • use percentages instead of fixed widths when thats not possible us viewport relative units VW, VH, Vmin,Vmax

  • use max-width instead of just width

  • set max-width of 100% on im,object,video, and iframe

  • use background-size:cover; to meet container sizes with images. warning this is not always wise due to scaling down to css in mobile designs.

  • We should be careful about using short hand options for future proofing such as background and background-color you may be introducing bugs by using shorthand though order of properties are important in this instance the shorthand will overwrite the longhand because they can contain extra properties.
